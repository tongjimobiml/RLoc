## Usage

How to run the code:

Start an experiment by:

``run ./Script/xxx.sh``

## Dataset

The dataset contains 2G and 4G telco data from two district: Jiading and Siping. The 2G telco data can be found in ``2g`` file and the 4G telco data can be found in ``4g`` file.
